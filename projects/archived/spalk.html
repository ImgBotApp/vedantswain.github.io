Spalk

Topic centric chatting with reputation based filtering

Aug, 2014 - Dec, 2014
  • View project on Github
  • Team Size: 5

Motivation

PChatting on anonymously on specific topic channels is not uncommon. However, when the physical world slowly shifted to the digital, there was an inflow of nuisance elements as well. This has lead to a lot of cases of trolling, harassment and annoyances.

Simply talking about what one likes with someone who also likes it may not always be a pleasant experience.

Solution

A topic-centric chat app with reputation based filters to ward off those who plan to ruin the hopes of possibly fine conversation. In order to establish such a system, every user must maintain a topic wise reputation. This reputation can be earned via peer appreciation during conversations.

Features

  • Categorised Topics

    Topics are indexed at the backend based on categories. This is specifically to make the search for new topics easier.

  • Trending Topics

    Topics undergo a temporal decay. When a topic is interacted with more regularly it climbs higher up the trending ladder.

  • Reputation Based Filtering

    A user may choose a reputation threshold (less than that of their own) for finding a match to converse with it. Users will only be paired if their reputation scores are above the thresholds set by each other.

    Additionally location based filtering may also be done. If this option is checked then the pairing of users will be further restricted to the userâ€™s cities.

  • Peer Approval

    Each user may score the other user during conversation in real time. This score given by another user is only revealed at the end of the conversation. This contributes to the overall score.

  • Profile Statistics

    Each user can view which topic they chat the most on. Along with which they can also see the topics they have maximum reputation on. Apart from which a topic wise reputation split and a total reputation is also presented.

Future

More Avenues to Gain Reputation

Currently peer approval is the only way to earn reputation points. Subsequent iterations of the app could consider other samaritan tasks to increase reputation. E.g. Filling in details for topics

Group Chat

Supporting multiple individuals on a single conversation can be explored to improve the reputation gain system. E.g. Approvals from all other individuals can lead to bonuses.
